Sub-Cells

The following Sub-Cells will be implemented and used:

  • Static Inverter
  • 2-input Domino AND
  • 3-input Domino AND
  • 2-input Domino OR
  • 3-input Domino OR
  • Propagate-Generate Unit - Produces Propagate and Generate signals for each bit of input
  • Carry Unit - Used Propagate and Generate signals to produce Carry signals for each input bit
  • Sum Unit - Determines the Sum for each bit, based on the Carry signal
  • Zero Unit - Signal goes HIGH if the calculation result is zero
  • Overflow Unit - Determines if Overflow has occurred in the calculation
